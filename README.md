# cccolide
supercollider + cprocessing combo

# dependencies
## required:
* GXX compiler
* cmake
* git (in system path)
* supercollider (binaries)

## automagically build from source:
* GLUT
* GL (not fully automated)
* cprocessing
* oscpack
* Openobject Quark

# building  (system must be online for the first time)

* ```git clone https://github.com/K0F/cccolide.git```
* ```cd cccolide```
* ```mkdir build```
* ```cmake ..```
* ```make -j5```
* ```./ccolide```

Enjoy!
